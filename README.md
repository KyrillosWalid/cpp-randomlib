cpp-randomlib
===========

Only C++ Header file gives random (integers - strings - floats - boolean)

It's extremely easy to setup. Just include **random.hpp** file in your code!

Get Random int
--------------

```c++
#include <iostream>
#include "random.hpp"

using namespace std;

int main(){
    cout<< randint() << "\n";
}
```

Get Random char
--------------

```c++
#include <iostream>
#include "random.hpp"

using namespace std;

int main(){
    cout<< randchar() << "\n";
}
```
Get Random char as string
--------------

```c++
#include <iostream>
#include "random.hpp"

using namespace std;

int main(){
    cout<< randcharstr() << "\n";
}
```

### What Can This Code Do for now ? 

- get random integers
- get random chars
- get random chars as string
